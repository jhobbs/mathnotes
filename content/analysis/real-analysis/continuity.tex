\section{Continuity}

\subsection{Limit Definition of Continuity}

Let $A$ be a set of real numbers and $f(x)$ a function $f : A \to \mathbb{R}.$ We say that $f(x)$ is \textbf{continuous} at $a$ if $\lim_{x \to a}{f(x)} = f(a).$

In other words, $f(a)$ exists, the limit of $f(x)$ as $x$ approaches $a$ exists, and these two values are equal.

\subsection{Epsilon-Delta Definition of Continuity}

This is an equivalent definition of continuity.

\emph{Theorem:} Let $A$ be a set of real numbers and $f(x)$ a function from $A$ into $\mathbb{R}.$ Assume also that $a$ is a real number and for every $\delta > 0,$ there exists numbers $x \in A$ satisfying $0 < \|x - a\| < \delta.$ Then $f(x)$ is continuous at $a$ if and only if for every $\epsilon > 0$ there exists $\delta > 0$ such that $\|f(x) - f(a)\| < \epsilon$ if $\|x - a\| < \delta$ and $x \in A.$

This is basically saying that we can choose as small of a change from $f(a)$ to $f(x)$ as we want ($\epsilon$) by picking a value of $x$ close enough to $a$ ($\delta.$)

\subsection{Continuity Rules}

\emph{Theorem:} Suppose $f(x)$ and $g(x)$ are continuous at $a,$ and $c$ is a real number. Then we have that:

\begin{enumerate}
\item $f(x) + cg(x)$ is continuous at $a.$
\end{enumerate}

\begin{enumerate}
\item $f(x)g(x)$ is continuous at $a$.
\end{enumerate}

\begin{enumerate}
\item $\frac{f(x)}{g(x)}$ is continuous at $a$ if $g(a) \neq 0.$
\end{enumerate}

\begin{enumerate}
\item $f(g(x))$ is continuous at $a.$
\end{enumerate}

A function is said to be continuous if it's continuous for all points in its domain. A function is said to be continuous on $[a, b]$ if it's continuous for all values in $(a, b),$ it's continuous from the right at $a$ and it's continuous from the left at $b.$

\subsection{More Theorems Related to Continuity}

\emph{Intermediate Value Theorem:} Suppose $f : [a, b] \to \mathbb{R}$ is continuous and that $r$ is a value between $f(a)$ and $f(b).$ Then there exists a number $c$ in $(a, b)$ for which $f(c) = r.$

\emph{Boundedness Theorem:} If $f$ is continuous on $[a, b]$ then it's bounded on $[a, b].$

\emph{Extereme Value Theorem:} If $f$ is continuous on $[a, b]$ then $f$ attains absolute minimum and absolume maximum values on $[a, b].$
